Welcome to the ``estim8`` documentation!
========================================

With ``estim8`` we bring model-based analysis closer to your research. The package provides simulation methods of ODE/DAE biorpocess models implemented in `Modelica <https://modelica.org/>`_ as `Functional Mockup units` (FMUs) combined with state of the art optimization algorithms, i.e. for parameter estimation.
